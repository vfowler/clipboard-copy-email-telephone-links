This prototype was inspired by The trouble with mailto email links and what to do instead by Adam Silver and Amy Hupe. Above, I offer choice by adding a clipboard copy button to email and telephone links. The links alone perform a primary action only, often fixed in the operating system.

  • Clicking an email link begins composing a new email message in the default mail app.
  • Clicking a telephone link dials the number to start a call in the default telephony app.

Providing a Copy to clipboard button opens alternative options. For example,

  • Use a non-default mail app to compose an email message
  • Send an SMS to the (mobile) phone number

Note, this script does not work in Internet Explorer. If you have a use case for clipboard copy buttons in this browser, please get in touch.
